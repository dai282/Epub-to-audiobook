## Installation

### 1. Install espeak-ng

**Linux (Ubuntu/Debian):**
```bash
sudo apt-get update
sudo apt-get install espeak-ng
```

**macOS:**
```bash
brew install espeak-ng
```

**Windows:**
1. Download the installer from [espeak-ng releases](https://github.com/espeak-ng/espeak-ng/releases)
2. Run the `.msi` installer
3. Add espeak-ng to your PATH
